# Cobrança

(Na ausência de um nome melhor) Cobrança é o documento que representa uma transação de crédito.

_Modelo_

```json
{
    "id":"9gfuh9ids89f3209f8ui0",
    "accounts":[
        {
            "role":"buyer",
            "accId":"jf8wejf934g398w58ugt",
            "value": -20
        },
        {
            "role":"receiver",
            "accId":"jf8wejf934g398w58ugt",
            "value": 19.95
        },
        {
            "role":"fee",
            "accId":"jf8wejf934g398w58ugt",
            "value": 0.05
        }
    ]
}
```
